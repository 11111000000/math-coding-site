# math-coding site

This is the rendered documentation site for [math-coding](https://github.com/11111000000/math-coding).

The site is generated by [math-coding-ui](https://github.com/11111000000/math-coding-ui) — a static site generator for math-coding convention projects.

## Source

The source content lives in [`math-coding-v0.618/`](./math-coding-v0.618/) as a submodule pointing to [math-coding/math-coding](https://github.com/11111000000/math-coding). Updates to convention flow here automatically when the submodule is bumped.

## Local development

```sh
# One-time: install math-coding-ui
npm install

# Build and serve locally with live reload
npm run dev
```

Open <http://localhost:4321/>.

## Build

```sh
npm run build
```

Output is in `dist/`. GitHub Pages deploys `dist/` on every push to `main` (see `.github/workflows/pages.yml`).

## Configuration

Edit `.math-coding/ui-config.yaml` to customise the site. See [math-coding-ui documentation](https://github.com/11111000000/math-coding-ui) for the full schema.

## Site structure

The generated site includes:

- `/` — landing page with packet statistics and lifecycle distribution
- `/axioms/` — the seven axioms with their statements
- `/theories/` — eight mathematical theories that ground the convention
- `/math/` — all packets with lifecycle badges
- `/graph/` — force-directed dependency graph
- `/fsm/` — six-state lifecycle FSM visualisation
- `/obsidian.html` — Obsidian compatibility notes
- `/changelog.html`, `/agents.html`, `/license.html` — manifest files
